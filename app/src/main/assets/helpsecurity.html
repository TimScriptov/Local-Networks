This version of the Serval Mesh Software contains improved security. We are now using Elliptic Curve Cryptography, which means messages and calls can only be opened by the device that has the right Serval ID (SID).
Ensuring the physical security of the device remains the responsibility of the user.
Upcoming releases will contain improvements to security options.
We have a list of all Android & Serval Permissions used, and what they are used for. However, the message that pops up on starting the Serval Mesh: "Serval Mesh has been granted superuser permissions", is required by Android to perform the necessary tasks, which are again explained in the Permissions List.
We also have a Privacy Policy.
