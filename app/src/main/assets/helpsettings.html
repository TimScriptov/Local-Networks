Serval Settings

There are five options on the Settings Screen.

Wifi Settings Wifi Settings allows you to choose the type of WiFi configuration you wish to use currently. This includes allowing Bluetooth and Allowing Wifi. These options can seriously affect your Power consumption.
Accounts Management Serval uses Accounts to provide initial identity management. More information is available at our Help - Account Management
View Logs A Developer option, to allow you to view issues that are logged.
Mesh Testing This tests the phone's chipset for Mesh support. This requires Root permissions. More information on issues with Mesh can be found on our Adhoc Page
MeshMS Settings MeshMS Settings allows you to change configuration for MeshMS
