Using the Main Screen

There are 9 buttons on the Main Screen.
There is also the display of your current Phone Number towards the bottom of the screen. This can be changed at any point in the Settings section.

The buttons on the Main Screen

Call Make calls via Serval Mesh
Messages Send messages via Serval Mesh
Contacts Manage contacts, access Address Book
Maps Serval Mapping application (if installed)
Share Files Share files on the Serval Mesh
Share Us Share our software with other people
Settings Settings for Serval Mesh application
Connect Enable Serval Mesh processes & connect to Networks
Help Guides, links, information
Accounts Information on personal settings
